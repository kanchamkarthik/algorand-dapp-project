# Algorand DApp Project

This repository is created for developing decentralized applications (DApps) using the Algorand blockchain.

## Objective
The goal of this project is to learn and implement Algorand blockchain development including smart contracts and decentralized applications.

## Technologies Used
- Algorand Blockchain
- Smart Contracts
- Python / JavaScript
- GitHub for version control

## Project Plan
1. Learn Algorand basics
2. Create a simple smart contract
3. Build a small decentralized application
4. Deploy on Algorand Testnet
